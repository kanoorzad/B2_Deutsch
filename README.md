# Deutsch Sprache – Wichtige Wörter – telc System – A1-B2

Build: v89

Changes:
- APP_VERSION is set to v1 and displayed as Flashcard Pro v1.
- Dynamic German-only synonyms are fetched live from OpenThesaurus.
- Verbs show up to 4 synonyms; nouns/adjectives/adverbs/other cards show up to 2.
- Synonyms display at the bottom of the flashcard with "(Synonyme)".
- Audio session keepalive is enhanced with Web Audio, silent audio loop, MediaSession, and speechSynthesis hook.

Notes:
- This app is a static HTML/CSS/JavaScript PWA, so the requested Python/Tkinter logic was implemented in JavaScript.
- OpenThesaurus requires internet access.
- Locked-screen browser TTS remains best-effort; guaranteed background playback needs a native app or bundled audio.
