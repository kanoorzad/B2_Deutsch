# Deutsch Sprache – Wichtige Wörter – telc System – A1-B2

Release: v2.0.4

Base:
- v2.0.3 accepted base

Fixes:
- Playback now pronounces only the requested verb forms per level:
  - A1/A2: Infinitiv, Präsens, Perfekt
  - B1: Infinitiv, Präsens, Präteritum, Perfekt
  - B1+/B2: Infinitiv, Präsens, Präteritum, Perfekt, Plusqp.
- B1+ and B2 verb forms on the flashcard now show:
  - Infinitiv + Präsens
  - Präteritum + Perfekt
  - Plusqp.
- Long verb form values are clipped/auto-fit inside the existing verb box without changing the card layout.
- A small top-left flashcard badge shows the current unit, for example `(Einheit-1)`.

Preserved:
- All 5123 cards
- All 8 translation languages
- Arabic and Kurmanji language buttons/interlinkage
- German interface
- German synonyms
- Persisch MP3-first audio logic
- Generated audio files and audio-manifest.js are not included and will not be overwritten
- Existing GitHub workflows are not changed
