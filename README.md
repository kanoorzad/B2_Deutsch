# B2 Native Flashcards iPhone App

Ready-to-host PWA for GitHub Pages.

## Included data
- B2 all 12 units: 1629 entries
- Irregular verb reference list: 150 verbs

## iPhone installation
1. Upload all files in this folder to a public GitHub repository.
2. Enable Settings > Pages > Deploy from branch > main > root.
3. Open the GitHub Pages URL in Safari on iPhone.
4. Tap Share > Add to Home Screen.

## Audio quality
The app uses iOS/browser speech voices and automatically requests:
- German: de-DE, slower rate
- English: en-US
- Dari: fa-AF, with fallback to Persian/Farsi voice if iOS has no Dari voice

True cloud AI voices require a paid TTS service or pre-generated MP3 files. This pack is prepared for browser-native high-quality voices without requiring a server.

## Data quality notes
- PDF irregular verb forms are loaded from the Akademie Deutsch irregular verb list.
- B2 translations come from your supplied B2 all 12 units list.
- Synonyms are shown as supplied alternatives/related meanings, not guaranteed dictionary-perfect synonyms.
