HOW TO INSTALL — B2_Deutsch (Persian audio, FREE-TIER paced build)
===================================================================
This zip mirrors your repo layout. Copy files keeping the SAME folders:
  • top-level files         -> repo ROOT (overwrite)
  • .github/workflows/*.yml -> repo .github/workflows/ (overwrite; TEST file is new)

WHAT CHANGED
------------
Your Azure key is the FREE (F0) tier, which is rate-limited to ~20 requests/min —
that's why every request got "429 THROTTLED". The generator now PACES itself at
~18 requests/min (REQUEST_INTERVAL=3.2s) so Azure stops throttling. It runs as a
SINGLE stream (parallel shards would re-trigger the limit).

RUN IT
------
1) (optional) Actions -> "TEST Persian audio" -> Run workflow. In ~1 min it should
   now show "SELFTEST: OK" and make ~11 clips (no more 429s).
2) Actions -> "Build Persian audio pack" -> Run workflow.
   • At ~18 clips/min, 8,405 clips take ~7.5h, so it needs about 2 runs.
   • It commits a checkpoint every 500 clips and skips clips that already exist,
     so when the first run ends, just RUN IT AGAIN to finish. Nothing is lost.
3) When it finishes with no missing clips, Persian audio is live.

WANT IT FAST INSTEAD?
---------------------
Upgrading the Azure Speech resource to the S0 (standard) tier removes the limit
(neural TTS includes 500,000 free chars/month on S0 — your whole pack is under
that, so likely free). On S0, set REQUEST_INTERVAL=0 in the workflow and it
finishes in minutes.
